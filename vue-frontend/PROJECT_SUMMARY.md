## 🎨 设计特色

### 1. 学术专业风格
- 主色调采用学术蓝 #1e40af，体现专业和可信
- 清晰的视觉层次，便于快速获取信息
- 简洁的设计，避免过多装饰干扰研究内容

### 2. 完整的设计令牌系统
- 60+ 个颜色变量
- 完整的间距系统（基于 8px 网格）
- 统一的圆角和阴影
- 一致的字体系统

### 3. 丰富的组件库
- 6 种按钮类型
- 4 种输入框变体
- 8 种标签状态
- 完整的布局组件

### 4. 响应式设计
- 6 个断点（xs, sm, md, lg, xl, 2xl）

- 颜色对比度符合 WCAG 标准

- 使用 CSS 变量，便于主题切换

## 💡 使用建议

1. **安装依赖**：
   ```bash
   cd vue-frontend
   npm install
   ```

2. **启动开发服务器**：
   ```bash
   npm run dev
   ```

3. **查看样式演示**：
   打开 `public/style-demo.html` 查看所有组件样式

4. **开始开发**：
   参考 `README.md` 和 `docs/STYLE_GUIDE.md` 开始开发
