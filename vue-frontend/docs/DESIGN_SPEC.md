# SmartPaper Vue 界面设计规范

## 1. 设计理念

### 1.1 设计定位
- **学术专业风格**：面向科研工作者的专业工具，简洁、清晰、高效
- **现代简约设计**：去除冗余装饰，聚焦内容本身
- **清晰的视觉层次**：通过色彩、间距、字体建立明确的层级关系

### 1.2 设计原则
1. **内容优先**：突出研究内容，减少视觉干扰
2. **一致性**：整个系统保持统一的视觉语言和交互模式
3. **可访问性**：确保文字对比度、交互区域大小符合无障碍标准
4. **响应式**：适配不同屏幕尺寸和设备
5. **性能优先**：快速加载，流畅交互

## 3. 字体系统

### 3.1 字体栈

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 'Oxygen',
             'Ubuntu', 'Cantarell', 'Fira Sans', 'Droid Sans', 'Helvetica Neue',
             sans-serif;
```

### 3.2 字体大小

```
超大标题（H1）:   2.5rem (40px)   - 页面主标题
大标题（H2）:    1.875rem (30px) - 区块标题
中标题（H3）:    1.5rem (24px)   - 组件标题
小标题（H4）:    1.25rem (20px)  - 子区块标题
正文（Body）:    1rem (16px)     - 正文内容
小正文:          0.875rem (14px) - 辅助说明
极小文字:        0.75rem (12px)  - 标签、脚注
```

### 3.3 字重

```
Bold (700):    主标题、重要操作
SemiBold (600): 次级标题、强调
Medium (500):  部分组件标题
Regular (400): 正文内容
Light (300):   辅助信息
```

### 3.4 行高

```
标题:   1.2   - 紧凑，用于大字号
正文:   1.6   - 舒适阅读
代码:   1.5   - 技术内容
```

## 4. 间距系统

基于 8px 网格系统：

```
xs (极小):  4px   - 紧密元素间距
sm (小):    8px   - 图标与文字间距
md (中):    16px  - 默认间距，组件内边距
lg (大):    24px  - 区块间距
xl (超大):  32px  - 大区块间距
2xl (特大): 48px  - 页面级间距
```

---

## 5. 圆角系统

```
sm (小):    4px   - 按钮、小卡片
md (中):    8px   - 输入框、标签
lg (大):    12px  - 卡片、面板
xl (超大):  16px  - 模态框、弹窗

```

## 6. 阴影系统

```
sm (小):    0 1px 2px 0 rgba(0, 0, 0, 0.05)
md (中):    0 4px 6px -1px rgba(0, 0, 0, 0.1), 0 2px 4px -1px rgba(0, 0, 0, 0.06)
lg (大):    0 10px 15px -3px rgba(0, 0, 0, 0.1), 0 4px 6px -2px rgba(0, 0, 0, 0.05)
xl (超大):  0 20px 25px -5px rgba(0, 0, 0, 0.1), 0 10px 10px -5px rgba(0, 0, 0, 0.04)
```

使用场景：
- sm: 小按钮、标签
- md: 卡片、下拉菜单
- lg: 模态框、抽屉
- xl: 重要弹窗、悬浮面板

## 7. 过渡与动画

### 7.1 基础过渡
```css
transition: all 0.2s ease-in-out;
```

### 7.2 动画时长

```
快速:  150ms - 频繁交互（按钮、输入框）
标准:  200ms - 大部分组件
慢速:  300ms - 模态框、页面切换
```

### 7.3 常用动画

```
淡入淡出:   opacity
上滑:      translateY(-20px) -> translateY(0)
下滑:      translateY(20px) -> translateY(0)
缩放:      scale(0.95) -> scale(1)
```

- 圆角: 12px
- 阴影: md

### 9.4 标签

- 内边距: 4px 12px

## 12. 可访问性规范

### 12.1 颜色对比度
- 正常文字: 至少 4.5:1
- 大文字（18px+）: 至少 3:1
- 图标: 至少 3:1

### 12.2 交互区域
- 最小点击区域: 44x44px
- 焦点可见: 明确的焦点样式

### 12.3 键盘导航
- Tab 顺序: 逻辑清晰
- 焦点陷阱: 模态框内
